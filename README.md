## Procesamiento de Ciencia de Datos


## Objetivo
Este proyecto tiene como objetivo aplicar técnicas básicas de control de versiones con Git y GitHub, y estructurar un flujo de trabajo reproducible para proyectos de Ciencia de Datos.

## Estructura básica del proyecto
preprocesamiento-ciencia-datos/
├── data/                # Carpeta para almacenar datos brutos y procesados  
├── notebooks/           # Jupyter Notebooks con análisis y pruebas  
├── scripts/             # Scripts Python para limpieza y transformación  
├── results/             # Resultados generados (gráficos, reportes, etc.)  
├── .gitignore           # Archivos y carpetas ignoradas por Git  
├── README.md            # Descripción general del proyecto
